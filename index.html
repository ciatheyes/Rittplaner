Rittplaner Reitgebiet um den Fannyhof in Altikon
Über diese Webseite

Der Fannyhof ist ein Reithof, der neben Heilpädagogischem Reiten auch gewöhnliche Ausreitgruppen für die ReiterInnen anbietet. Je nach Wetter, Jahreszeit, Anzahl ReiterInnen etc. gibt es verschiedenste Aktivitäten, die auf so einem Ausritt unternommen werden können. Eine Auswahl dieser Aktivitäten sollen mithilfe der Karte auf dieser Webseite erläutert werden.
Zudem besteht die Möglichkeit, einen Rundweg vom Hof aus zu einer Auswahl von Aktivitäten zu berechnen.



Navigation - Wie bediene ich diese Karte?

Jeder rote Pin auf der Karte stellt eine Aktivität dar. Klickt man darauf, werden im Informations-Tab auf der linken Seite Informationen zur angewählten Aktivität aufgeführt.
Sind mehrere Aktivitäten nahe beieinander, so werden sie als roter Kreis dargestellt. Die Anzahl der damit zusammengefassten Aktivitäten steht dann in der Mitte des Kreises.

Die Aktivitäten sind verschiedenen Kategorien zugeordnet. Diese sind im Kategorien-Tab aufgelistet. Klickt man auf eine Kategorie und entfernt damit das Häckchen, so werden auf der Karte die Aktivitäten dieser Kategorie entfernt. Setzt man das Häckchen wieder, werden sie auf der Karte wieder hinzugefügt.
Eine Aktivität kann mehreren Kategorien zugeschrieben sein.

Schliesslich kann über das Wegberechnungs-Tab der kürzeste Weg zu einer Auswahl von Aktivitäten berechnet und auf der Karte dargestellt werden. Damit man einen Weg berechnen kann muss mindestens eine Aktivität ausgewählt werden (durch Anklicken auf der Karte. Ausserdem kann man angeben, welche Prozentanteile des Rittes man im Schritt, Trab und Gallop verbringt und jeweils mit welcher Geschwindigkeit. Für diese Werte können aber auche infach die Vorgabewerte verwendet werden.

Die verschiedenen Tabs sind über die Navigationsspalte ganz links erreichbar und können mit dem Pfeil oben rechts neben der Überschrift wieder geschlossen werden.

Tipp:
Da eine Aktivität mehreren Kategorien angehören kann, muss man eine Kategorie zuerst ausblenden und dann wieder einblenden, um sicher zu stellen will, dass wirklich alle Aktivitäten dieser Kategorie angezeigt werden.
